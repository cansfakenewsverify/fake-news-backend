# AI-Driven Anti-Scam System (Backend)

這是防詐系統的後端 API 專案，使用 FastAPI 框架建置。

## 開發環境建置指南

1. **複製專案**
   `git clone https://github.com/你的帳號/anti-scam-backend.git`
   `cd anti-scam-backend`

2. **建立並啟動虛擬環境**
   `python -m venv venv`
   `.\venv\Scripts\activate`

3. **安裝套件**
   `pip install -r requirements.txt`

4. **啟動伺服器**
   `uvicorn main:app --reload`
